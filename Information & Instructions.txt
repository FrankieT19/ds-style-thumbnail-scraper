[INTRO]

Thank you for downloading DS Style Thumbnail Scraper v1.3.

For DS Style v6.6 or newer.

This tool creates thumbnail artwork for DS Style on the EZ-Flash Omega and Omega Definitive Edition.

#####################################################################################

[STANDARD THUMBNAIL PACKS]

- 120 x 80 artwork is saved into IMGS
- 80 x 80 artwork is saved into IMGS2

#####################################################################################

[BUILD PACK]

The Build Pack tab creates a complete thumbnail pack without needing your ROM files.

Choose a thumbnail size, then choose an artwork source:
- Libretro downloads artwork from the online Libretro library. You can use box art, gameplay screenshots, or title screens.
- Local Pack uses an unzipped artwork collection already stored on your computer.

Choose which game regions to include. Artwork priority controls which regional artwork is preferred when more than one version is available.

The crop controls apply to the complete pack. Use Search & Preview and Exceptions if individual games need different artwork or positioning.

Press Build Thumbnail Pack when ready. Finished packs are saved in a dated folder inside DS Style Thumbnail Scraper output, beside the program.

#####################################################################################

[CUSTOM ART]

Use the Custom Art tab to give custom artwork to any file or folder.

For files, Custom Art uses the filename without its extension. It does not use the internal game header, so you do not need to edit or patch ROM files.

For folders, Custom Art uses the folder name exactly as it appears on your SD card.

File example:
If your file is named:
Pokemon Emerald Rogue.gba

The custom thumbnail should be named:
Pokemon Emerald Rogue.bmp

Files with the same name but different extensions share artwork. For example, Game.gba and Game.gbc both use Game.bmp.

Names otherwise remain exact. Game (1) and Game (2) use Game (1).bmp and Game (2).bmp.

Folder example:
If your folder is named:
Game Boy Advance

The custom thumbnail should be named:
Game Boy Advance.bmp

The scraper will place it into:
- IMGS/CUSTOM for 120 x 80 title thumbnails
- IMGS2/CUSTOM for 80 x 80 box thumbnails

Custom artwork matching is not case-sensitive.

Due to the GBA's RAM limitations, each CUSTOM folder can contain up to 256 images.

Older DS Style versions do not use the CUSTOM artwork folders, so this feature needs DS Style v6.6 or newer.

#####################################################################################

[NOTES]

The tool does not download games and does not edit your ROM files.
